# FM Website

A lightweight static website for an FM-style audio experience. The project has no build step and can be opened directly in a browser or served from any static host.

## Project Structure

- `index.html` - page markup
- `src/styles.css` - responsive visual design
- `src/main.js` - player controls and station switching

## Local Preview

Open `index.html` in a browser, or run a local static server:

```sh
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deployment

This project is ready for GitHub Pages or any static hosting provider.
